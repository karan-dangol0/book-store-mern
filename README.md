# book-store-mern

## Setup

1. clone the project using git clone url

## backend Setup

1. open the backend directory cd backend
2. make .env file
code `
PORT=3000
MONGO_URI=yourmongouri
`
3. install all pakages using npm i
4. npm run dev
   
   

## Frontend setup 
1. Open the frontend directory cd frontend
2. install all pakages using npm i
3. Run the frontend using npm run dev
4. Go to localhost:5173

Setup complete, Happy coding!
